## 🚀 How to Launch

### Option 1: Hostinger
1. Login to Hostinger → Go to **File Manager**
2. Open `public_html` folder
3. Upload `index.html` (drag & drop)
4. Your site is LIVE at your domain!

### Option 2: GitHub Pages (Free Hosting)
1. Go to github.com → Create new repository (name: `yourchoice` or any name)
2. Upload `index.html` to the repo
3. Go to repo **Settings → Pages**
4. Source: select `main` branch → Save
5. Your site goes live at: `https://yourusername.github.io/yourchoice`

### Option 3: Netlify (Free, Recommended)
1. Go to netlify.com → Sign up free
2. Drag & drop the entire `yourchoice-website` folder
3. Site is live instantly with a free URL!

## 📧 IMPORTANT: Set Your Email for Orders

Open `index.html` in any text editor (Notepad, VS Code etc.)

Search for: `your-email@gmail.com`

Replace with your actual email address.

This ensures every order triggers an email to you.
